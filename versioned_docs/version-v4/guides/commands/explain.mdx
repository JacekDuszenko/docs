:::info

We send information to OpenAI about the pull request, including parts of the git patch and git diff.
We do not send any information that is not retrieved from the pull request.
For more questions, contact us on [discord](https://reviewpad.com/discord).

:::

The `explain` command adds an inline comment explaining the selected piece of code.
If the line comment is created on a function name, the overall body of the function is explained.
If a multi-line comment is created, the overall block of code is explained.

**Syntax**:

```shell
/reviewpad explain
```
