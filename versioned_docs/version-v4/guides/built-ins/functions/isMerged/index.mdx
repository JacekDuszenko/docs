:::caution

This built-in is unavailable for issues. Reviewpad will ignored it when running on issues.

:::

**Description**:

Verifies whether a pull request is merged, returning `true` or `false` as appropriate.

**Parameters**:

_none_

**Return value**:

| Type      | Description                                              |
| :-------- | :--------------------------------------------------------|
| `boolean` | `true` if the pull request is merged, `false` otherwise. |

**Examples**:

```yml
$isMerged()
```
