**Description**:

Retrieves the total number of approvals for a pull request.

**Parameters**:

_none_

**Return value**:

| Type  | Description                                                  |
| :---- | :----------------------------------------------------------- |
| `int` | The total number of approvals the pull request has received. |

**Examples**:

```yml
$approvalsCount()
```
