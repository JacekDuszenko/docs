**Description**:

Verifies if the patch contains a symbol with the provided annotation, returning `true` or `false` as appropriate.

:::info

Please note that for this to work a symbol must be annotated with the `@reviewpad-an <annotation>` annotation.

For more information annotations, please see [this page](./annotations).

:::

:::caution

This built-in is not available for issues.

:::

**Parameters**:

| Parameter    | Type   | Description                              |
| :----------- | :----- | :--------------------------------------- |
| `annotation` | string | The annotation to look for in the patch. |

**Return value**:

| Type      | Description                                                                           |
| :-------- | :------------------------------------------------------------------------------------ |
| `boolean` | `true` if the patch changes a symbol with the provided annotation, `false` otherwise. |

**Examples**:

```yml
$hasAnnotation("critical")
```
