**Description**:

Retrieves the milestone title associated with the pull request.

**Parameters**:

_none_

**Return value**:

| Type     | Description                                           |
| :------- | :---------------------------------------------------- |
| `string` | The milestone title associated with the pull request. |

**Examples**:

```yml
$milestone()
```
