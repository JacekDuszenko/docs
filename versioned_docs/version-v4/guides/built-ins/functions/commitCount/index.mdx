This built-in is unavailable for issues. Reviewpad will ignored it when running on issues.

:::

**Description**:

Retrieves the total number of commits made into the pull request.

**Parameters**:

_none_

**Return value**:

| Type  | Description                                      |
| :---- | :----------------------------------------------- |
| `int` | The total number of commits in the pull request. |

**Examples**:

```yml
$commitCount()
```
