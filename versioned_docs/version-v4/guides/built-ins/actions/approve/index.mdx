:::caution

This built-in is unavailable for issues. Reviewpad will ignored it when running on issues.

:::

:::note note

This built-in is a shortcut to the [`review`](./built-ins#review) built-in.
It is the same as `$review("APPROVE", <MESSAGE>)`.

:::

**Description**:

Submits a pull request review marked as approved.


**Parameters**:

| Parameter            | Type    | Description                        |
| :------------------- | :------ | :--------------------------------- |
| `message` (optional) | string  | The message to write as a comment. |

**Examples**:

```yml
$approve()
```
