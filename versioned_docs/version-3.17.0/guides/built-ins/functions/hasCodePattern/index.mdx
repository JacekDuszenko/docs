**Description**:

Verifies if the patch matches the provided code pattern, returning `true` or `false` as appropriate.

The code pattern needs to be a compilable regular expression.

:::caution

This buit-in is **not** available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

| Parameter       | Type   | Description                                 |
| :-------------- | :----- | :------------------------------------------ |
| `query_pattern` | string | The query pattern to look for in the patch. |

**Return value**:

| Type      | Description                                                       |
| :-------- | :---------------------------------------------------------------- |
| `boolean` | `true` if the patch matches the query pattern, `false` otherwise. |

**Examples**:

```yml
$hasCodePattern("placeBet\(.*\)")
```
