**Description**:

Fails the action with a given fail message which will appear in the logs of the GitHub action.

**Available for:**

| Type           | Available |
| :------------- | :-------- |
| `issue`        | ✅        |
| `pull_request` | ✅        |

**Parameters**:

| Parameter       | Type    | Description        |
| :-------------- | :------ | :----------------- |
| `fail_message`  | string  | The fail message.  |

**Examples**:

```yml
$fail("missing specs")
```
