**Description**:

Adds a message to the info section of the report.

**Available for:**

| Type           | Available |
| :------------- | :-------- |
| `issue`        | ✅        |
| `pull_request` | ✅        |

**Parameters**:

| Parameter  | Type    | Description                                   |
| :--------- | :------ | :-------------------------------------------- |
| `comment`  | string  | The comment to be added in the informations.  |

**Examples**:

```yml
$info("Please do not forget to add the tests.")
```
