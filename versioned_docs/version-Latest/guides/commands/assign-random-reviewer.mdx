The `assign-random-reviewer` command triggers the [`$assignRandomReviewer`](../guides/built-ins#assignrandomreviewer) action.

This command assigns a random user of the GitHub organization as the reviewer.

**Arguments**:

_none_

**Syntax**:

```shell
/reviewpad assign-random-reviewer
```

**Examples**:

```shell
/reviewpad assign-random-reviewer
```
