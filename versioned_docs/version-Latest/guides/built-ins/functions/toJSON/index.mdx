**Description**:

Converts a JSON serialized string to a JSON object.

**Parameters**:

| Parameter     | Type   | Description            |
| :------------ | :----- | :--------------------- |
| `json_string` | string | JSON serialized string |

**Return value**:

| Type   | Description                                             |
| :----- | :------------------------------------------------------ |
| `JSON` | The JSON object corresponding to the serialized string. |

**Examples**:

```yml
$toJSON($context())
$toJSON("[1, 2, 3]")
```
