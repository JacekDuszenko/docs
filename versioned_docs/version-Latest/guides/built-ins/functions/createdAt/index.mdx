**Description**:

Retrieves the time the pull request / issue was created at.

**Parameters**:

_none_

**Return value**:

| Type    | Description                                              |
| :------ | :------------------------------------------------------- |
| `int64` | The number of seconds elapsed since January 1, 1970 UTC. |

**Examples**:

