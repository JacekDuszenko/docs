**Description**:

Retrieves the total number of commits made into the pull request.

:::caution

This buit-in is **not** available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type  | Description                                      |
| :---- | :----------------------------------------------- |
| `int` | The total number of commits in the pull request. |

**Examples**:

```yml
$commitCount()
```
