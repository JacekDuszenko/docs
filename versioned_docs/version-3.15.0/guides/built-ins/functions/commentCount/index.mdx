**Description**:

Retrieves the total number of comments made into the pull request.

**Available for:**

| Type           | Available |
| :------------- | :-------- |
| `issue`        | ✅        |
| `pull_request` | ✅        |

**Parameters**:

_none_

**Return value**:

| Type  | Description                                       |
| :---- | :------------------------------------------------ |
| `int` | The total number of comments in the pull request. |

**Examples**:

```yml
$commentCount()
```
