**Description**:

Checks if the issue is linked to a GitHub project with a given title.

If the project doesn't exist, an error is returned.

**Available for:**

| Type      | Available |
| :-------- | :-------- |
| `issue`   | ✅        |
| `pull_request` | ✅   |

**Parameters**:

| Parameter       | Type    | Description                                |
| :-------------- | :------ | :----------------------------------------- |
| `project_title` | string  | The title of the project - case sensitive. |

**Return value**:

| Type      | Description                                                       |
| :-------- | :---------------------------------------------------------------- |
| `boolean` | `true` if the issue is linked to the project, `false` otherwise.  |

**Examples**:

```yml
$isLinkedToProject("project title")
```
