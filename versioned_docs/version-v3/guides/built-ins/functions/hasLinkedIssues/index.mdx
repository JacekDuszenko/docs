**Description**:

Checks if a pull request has associated issues that might be closed by it.

:::caution

This built-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type      | Description                                                      |
| :-------- | :--------------------------------------------------------------- |
| `boolean` | `true` if the pull request has linked issues, `false` otherwise. |

**Examples**:

```yml
$hasLinkedIssues()
```
