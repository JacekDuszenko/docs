**Description**:

Verifies whether a pull request is merged, returning `true` or `false` as appropriate.

:::caution

This built-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type      | Description                                              |
| :-------- | :--------------------------------------------------------|
| `boolean` | `true` if the pull request is merged, `false` otherwise. |

**Examples**:

```yml
$isMerged()
```
