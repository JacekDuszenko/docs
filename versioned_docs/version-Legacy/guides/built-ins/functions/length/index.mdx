**Description**:

Retrieves the length of an array.

**Parameters**:

| Parameter | Type      | Description            |
| :-------- | :-------- | :--------------------- |
| `array`   | \[]string | The array of elements. |

**Return value**:

| Type  | Description              |
| :---- | :----------------------- |
| `int` | The length of the array. |

**Examples**:

```yml
$length(["a", "b"]) # 2
```
