**Description**:

Retrieves the list of commit messages of the pull request.

:::caution

This built-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type       | Description                                      |
| :--------- | :----------------------------------------------- |
| `[]string` | The list of commit messages of the pull request. |

**Examples**:

```yml
$commits()
```
