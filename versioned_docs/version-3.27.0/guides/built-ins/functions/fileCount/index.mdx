**Description**:

Retrieves the total number of files changed in the patch.

:::caution

This built-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type  | Description                                     |
| :---- | :---------------------------------------------- |
| `int` | The total number of files changed in the patch. |

**Examples**:

```yml
$fileCount()
```
