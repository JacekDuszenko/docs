**Description**:

Submits a pull request review marked as approved.

:::note note

This built-in is a shortcut to the [`review`](/guides/built-ins#review) built-in.
It is the same as `$review("APPROVE", <MESSAGE>)`.

:::

:::caution

This built-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

| Parameter            | Type    | Description                        |
| :------------------- | :------ | :--------------------------------- |
| `message` (optional) | string  | The message to write as a comment. |

**Examples**:

```yml
$approve()
```
