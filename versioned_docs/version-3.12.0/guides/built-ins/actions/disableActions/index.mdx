**Description**:

Disables the list of actions passed as an argument.

**Available for:**

| Type           | Available |
| :------------- | :-------- |
| `issue`        | ✅        |
| `pull_request` | ✅        |

**Parameters**:

| Parameter | Type      | Description                         |
| --------- | --------- | ----------------------------------- |
| `actions` | \[]string | The list of actions to be disabled. |

**Examples**:

```yml
$disableActions(["assignReviewer"])
```
