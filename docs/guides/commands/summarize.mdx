This is a command that is currently in beta phase.

We send information to OpenAI about the pull request, including parts of the git patch and git diff.
We do not send any information that is not retrieved from the pull request.
For more questions, contact us on [discord](https://reviewpad.com/discord).

:::info

You can also use the [`summarize` as a built-in](/guides/built-ins/#summarize) on your `reviewpad.yml` configuration.

:::

The `summarize` command adds a comment to the pull request with the summary of the changes.

**Arguments**:

_none_

**Syntax**:

```shell
/reviewpad summarize
```

**Examples**:

```shell
/reviewpad summarize
```
