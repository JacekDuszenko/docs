:::info

You can also use the [`assignCodeAuthorReviewers` as a built-in](/guides/built-ins#assignCodeAuthorReviewers) on your `reviewpad.yml` configuration.

:::

The `assign-reviewer` command triggers the [`$assignCodeAuthorReviewers`](/guides/built-ins#assigncodeauthorreviewers) action.

This command assigns a defined amount of reviewers to the pull request based on code author information.

**Arguments**:

| Arguments                   | Description                        |
| :-------------------------- | :--------------------------------- |
| `total_required_reviewers`  | Total number of required reviewers |

**Syntax**:

```shell
/reviewpad assign-reviewer <total_required_reviewers>
```

**Examples**:

```shell
/reviewpad assign-reviewer
```

```shell
/reviewpad assign-reviewer 1
```

```shell
/reviewpad assign-reviewer 5
```
