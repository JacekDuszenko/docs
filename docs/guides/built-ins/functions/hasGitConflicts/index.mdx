**Description**:

Determines if the pull request has Git conflicts, returning `true` or `false` as appropriate.

:::caution

This buit-in is **not** available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type      | Description                                                      |
| :-------- | :--------------------------------------------------------------- |
| `boolean` | `true` if the pull request has Git conflicts, `false` otherwise. |

**Examples**:

```yml
$hasGitConflicts()
```
