**Description**:

Retrieves the list of all file paths changed in the pull request.

:::caution

This buit-in is **not** available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type       | Description                                          |
| :--------- | :--------------------------------------------------- |
| `[]string` | A list of all file paths changed files in the patch. |

**Examples**:

```yml
$filesPath()
```
