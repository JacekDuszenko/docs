:::caution

This built-in is unavailable for issues. Reviewpad will ignored it when running on issues.

:::

**Description**:

Retrieves the list of all file paths changed in the pull request.

**Parameters**:

_none_

**Return value**:

| Type       | Description                                          |
| :--------- | :--------------------------------------------------- |
| `[]string` | A list of all file paths changed files in the patch. |

**Examples**:

```yml
$filesPath()
```
