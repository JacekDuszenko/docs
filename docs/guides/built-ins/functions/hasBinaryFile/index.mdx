This built-in is not available for issues.

:::

**Description**:

Checks if a pull request has a binary file in the patch.

**Parameters**:

_none_

**Return value**:

| Type      | Description                                                           |
| :-------- | :-------------------------------------------------------------------- |
| `boolean` | `true` if the pull request contains a binary file, `false` otherwise. |

**Examples**:

```yml
$hasBinaryFile()
```
