**Description**:

Retrieves the type of the
- [pull request event](https://docs.github.com/en/actions/using-workflows/events-that-trigger-workflows#pull_request) when Reviewpad is triggered by a pull request event.
- [issue event](https://docs.github.com/en/actions/using-workflows/events-that-trigger-workflows#issues) when Reviewpad is triggered by an issue event.


**Parameters**:

_none_

**Return value**:

| Type     | Description                                     |
| :------- | :---------------------------------------------- |
| `string` | The `pull_request`/`issue` event activity type. |

**Examples**:

```yml
$eventType()
```
