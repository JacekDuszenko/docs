**Description**:

Retrieves the name of the branch the pull request should be pulled into.

:::caution

This built-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type     | Description                                                    |
| :------- | :------------------------------------------------------------- |
| `string` | The name of the branch the pull request should be pulled into. |

**Examples**:

```yml
$base()
```
