**Description**:

Retrieves the state of a pull request / issue.
This state can be either `open` or `closed`.

**Parameters**:

_none_

**Return value**:

| Type     | Description                                                                 |
| :------- | :-------------------------------------------------------------------------- |
| `string` | The state of the pull request / issue. The state can be `open` or `closed`. |

**Examples**:

```yml
$state()
```
