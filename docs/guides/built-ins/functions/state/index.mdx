**Description**:

Retrieves the state of an issue or pull request.
This state can be either `open` or `closed`.

**Parameters**:

_none_

**Return value**:

| Type     | Description                                                               |
| :------- | :------------------------------------------------------------------------ |
| `string` | The state of the issue/pull request. The state can be `open` or `closed`. |

**Examples**:

```yml
$state()
```
