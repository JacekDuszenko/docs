**Description**:

Retrieves the status of a workflow run.

**Parameters**:

| Parameter       | Type   | Description               |
| :-------------- | :----- | :------------------------ |
| `workflow_name` | string | The name of the workflow. |

**Return value**:

| Type     | Description                     |
| :------- | :------------------------------ |
| `string` | The status of the workflow run. |

**Examples**:

```yml
$workflowStatus("reviewpad")
```
