**Description**:

Converts a string into a number.

**Parameters**:

| Parameter | Type   | Description                              |
| :-------- | :----- | :--------------------------------------- |
| `string`  | string | The string value to convert to a number. |

**Return value**:

| Type  | Description          |
| :---- | :------------------- |
| `int` | The converted number |

**Examples**:

```yml
$toNumber("1")
$toNumber("-1")
$toNumber($selectFromContext("$.comments"))
```
