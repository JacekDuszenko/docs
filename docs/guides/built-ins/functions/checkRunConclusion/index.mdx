**Description**:

Retrieves the conclusion of the given [check run](https://docs.github.com/en/rest/guides/getting-started-with-the-checks-api#about-check-runs).

:::caution

This buit-in is **not** avaible for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

| Parameter | Type   | Description                |
| :-------- | :----- | :------------------------- |
| `name`    | string | The name of the check run. |

**Return value**:

| Type     | Description                                                                                                                                  |
| :------- | :------------------------------------------------------------------------------------------------------------------------------------------- |
| `string` | The conclusion of the check run, can be `action_required`, `cancelled`, `failure`, `neutral`, `success`, `skipped`, `stale`, or `timed_out`. |

**Examples**:

```yml
$checkRunConclusion("build")
```
