**Description**:

Retrieves the timestamp of the last event in the timeline.

**Parameters**:

_none_

**Return value**:

| Type    | Description                                              |
| :------ | :------------------------------------------------------- |
| `int64` | The number of seconds elapsed since January 1, 1970 UTC. |

**Examples**:

