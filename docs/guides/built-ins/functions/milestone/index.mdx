**Description**:

Retrieves the milestone title associated to the pull request.

:::caution

This buit-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Return value**:

| Type     | Description                                         |
| :------- | :-------------------------------------------------- |
| `string` | The milestone title associated to the pull request. |

**Examples**:

```yml
$milestone()
```
