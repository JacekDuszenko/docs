**Description**:

Fails the Reviewpad check run status with a custom message without failing Reviewpad.

This built-in is especially useful to disable the merge action when some conditions are not met.
Disabling the Merge action on GitHub can be done with help of a branch protection rule.
This branch protection rule must require the Reviewpad check run status to succeed before merging.

The custom message is added to the Reviewpad status check summary.

**Parameters**:

| Parameter       | Type    | Description        |
| :-------------- | :------ | :----------------- |
| `fail_message`  | string  | The fail message.  |

**Examples**:

```yml
$failCheckStatus("One approval from the security team is missing.")
```
