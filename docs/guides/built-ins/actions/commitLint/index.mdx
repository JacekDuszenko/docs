**Description**:

Checks if the commits in the pull request follow the [conventional commits specification](https://www.conventionalcommits.org/en/v1.0.0/).

:::caution

This built-in is not available for [`issue`](/guides/syntax#workflow).

:::

**Parameters**:

_none_

**Examples**:

```yml
$commitLint()
```
