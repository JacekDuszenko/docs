**Description**:

Adds a pull request to a project with a particular status.

If the project doesn't exist, an error is returned.

If the status doesn't exist, an error is returned.

**Available for:**

| Type           | Available |
| :------------- | :-------- |
| `issue`        | ✅        |
| `pull_request` | ✅        |

**Parameters**:

| Parameter       | Type    | Description                                              |
| :-------------- | :------ | :------------------------------------------------------- |
| `project_title` | string  | The title of the project  - case sensitive.              |
| `status`        | string  | The status of the pull request (must exist as a status). |

**Examples**:

```yml
$addToProject("reviewpad", "in progress")
```
